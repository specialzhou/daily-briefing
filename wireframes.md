## 🎯 交互流程

### 1. 标记已读流程
```
用户点击 [✓已读] 
    ↓
卡片变灰色，标题加删除线
    ↓
状态保存到 LocalStorage
    ↓
归档时自动归类到已读
```

### 2. 收藏流程
```
用户点击 [☆收藏]
    ↓
☆ → ★ (实心)
    ↓
内容添加到收藏列表
    ↓
可在"收藏"页面查看
```

### 3. 笔记流程
```
用户点击 [📝笔记]
    ↓
弹出笔记对话框
    ↓
用户输入 Markdown 内容
    ↓
保存 → 显示笔记图标
    ↓
可编辑/删除
```

### 4. 任务转化流程
```
用户点击 [✓任务]
    ↓
弹出任务设置
  • 任务标题 (默认内容标题)
  • 优先级 (高/中/低)
  • 截止日期
    ↓
确认 → 添加到任务列表
    ↓
可追踪状态 (待办/进行中/已完成)
```
